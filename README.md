# hxtskills

Collection of AI skills for various applications.

## Available Skills

### generate-genetic-review

**English Description:**
A skill for generating comprehensive literature reviews for genetic research, rare diseases, and genomic sequencing. This skill enables the systematic creation of high-quality, academic literature reviews focusing on genetic disorders, genomic technologies, and clinical applications. Features include systematic literature search strategy, structured manuscript organization, genetic terminology standards, statistical reporting guidelines, quality assurance protocols, and integration with Google NotebookLM for PDF processing.

**中文描述:**
一个用于生成遗传学研究、罕见疾病和基因组测序综合文献综述的技能。该技能能够系统性地创建高质量的学术文献综述，重点关注遗传性疾病、基因组技术和临床应用。功能包括系统性文献检索策略、结构化文稿组织、遗传学术语标准、统计数据报告指南、质量保证协议，以及与Google NotebookLM集成进行PDF处理。

**Key Features / 主要功能:**
- Systematic literature search strategy / 系统性文献检索策略
- Structured manuscript organization / 结构化文稿组织
- Genetic terminology standards / 遗传学术语标准
- Statistical reporting guidelines / 统计数据报告指南
- Quality assurance protocols / 质量保证协议
- Google NotebookLM integration for PDF processing / 与Google NotebookLM集成进行PDF处理
- Sequential writing approach (Title → Abstract → Introduction → Methods → Results → Discussion → Conclusion) / 顺序写作方法（标题→摘要→引言→方法→结果→讨论→结论）
- Data summary tables with 1-2 key tables / 数据汇总表（包含1-2个关键表格）
- Figure descriptions with 2-3 detailed captions / 图表说明（包含2-3个详细图注）
- Proper citation of PDF-derived content / 对PDF来源内容的适当引用

**Usage / 使用场景:**
Trigger when requesting genetic literature reviews, rare disease surveys, genomic medicine overviews, gene therapy assessments, or inherited disorder analyses.
在请求遗传文献综述、罕见疾病调查、基因组医学概述、基因治疗评估或遗传性疾病分析时使用。
