# 自适应遗传文献综述生成技能

此技能可生成针对三个不同原型自适应调整的高质量遗传学文献综述，基于研究目的调整其方法和重点。

## 目的
生成高质量、学术性的遗传学文献综述，重点关注：
- 遗传性疾病和罕见疾病
- 基因组测序技术
- 遗传性疾病
- 群体遗传学
- 基因组医学
- 基因治疗与编辑

综述可适应三种原型之一：
- **A型（范式/伦理）**：政策、伦理、新标准
- **B型（临床管理）**：筛查、指南、实践
- **C型（精准机制）**：遗传学、罕见病、靶点

## 范围
- 30-80篇参考文献
- 4000-8000字
- 学术写作标准
- 临床相关性
- 批判性分析
- 与Google NotebookLM集成处理PDF
- 基于研究目的的自适应方法

## 主要特性
- 基于研究目的的自适应综述类型选择（A/B/C）
- 针对综述类型的系统文献检索策略
- 具有类型特定部分的结构化手稿组织
- 遗传学术语标准
- 按类型划分的统计报告指南
- 质量保证协议
- 用于PDF处理的Google NotebookLM集成
- 类型特定的写作方法（范式/伦理、临床管理或精准机制）
- 类型适当的数据汇总表和图表
- 正确引用PDF派生内容

## 综述类型

### A型：范式与决策（政策/伦理重点）
- **参考范式**: PES/PGT-P综述 [PMC11369226]
- **适用对象**: 新技术争议、伦理讨论、政策制定、建立新标准
- **核心输出**: 利益相关者分析表、获益-风险评估模型、政策建议
- **语调**: 批判性、平衡性、前瞻性
- **重点**: 伦理含义、社会影响、政策考虑

### B型：临床管理（实践/指南重点）
- **参考范式**: BRCA携带者综述 [PMC12758042]
- **适用对象**: 疾病诊断/治疗指南、筛查策略、预防干预、临床实施障碍
- **核心输出**: 风险分层算法、临床管理路径、实施障碍分析
- **语调**: 规定性、实用性、循证性
- **重点**: 临床实施、实践指南、结果优化

### C型：精准医学（机制/靶点重点）
- **参考范式**: CANDLE/SOD1-ALS综述 [PMC12744253]
- **适用对象**: 单基因疾病、罕见病、分子机制、靶向药物开发
- **核心输出**: 基因型-表型关联矩阵、分子通路图、病例级系统综述
- **语调**: 机制性、精确性、分析性
- **重点**: 分子机制、基因型-表型关联、靶向治疗

## PDF集成
- 通过Google NotebookLM上传和处理提供的PDF
- 从PDF提取参考文献和数据
- 将PDF发现与数据库文献整合
- 维护所有来源的正确引用标准
- 根据所选综述类型调整PDF集成方法

## 按类型顺序写作方法
1. 战略一致性（综述类型选择）
2. 目标文献挖掘（类型特定搜索）
3. 结构蓝图（类型特定大纲）
4. 高密度起草（类型适当内容）
5. 视觉综合（类型特定图表）
6. 最终润色（类型一致的综述）

## 包含的文件
- `SKILL.md`: 主要技能定义和自适应指南
- `CLAUDE.md`: 类型特定的写作指南和标准
- `IMPLEMENTATION_PLAN.md`: 逐步执行计划
- `WORKFLOW.md`: 按类型的阶段性系统过程
- `DOMAINS.md`: 遗传研究领域分类
- `GLOSSARY.md`: 全面的遗传学术语表
- `PDF_REFERENCES.md`: PDF集成和引用指南
